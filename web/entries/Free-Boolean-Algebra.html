Free Boolean Algebra

Brian Huffman üåê

March 29, 2010

Abstract

This theory defines a type constructor representing the free Boolean algebra over a set of generators. Values of type (Œ±)formula represent propositional formulas with uninterpreted variables from type Œ±, ordered by implication. In addition to all the standard Boolean algebra operations, the library also provides a function for building homomorphisms to any other Boolean algebra type.
