Referring to AFP Entries

 

Once you have downloaded the AFP, you can include its articles and theories in your own developments. If you would like to make your work available to others /without/ having to include the AFP articles you depend on, here is how to do it.

Let PATH be the absolute path where you have put the unpacked AFP directory that you downloaded. Add PATH on a separate line to the file ~/.isabelle/IsabelleXYZ/etc/components, where XYZ is the Isabelle release you are working with, for example XYZ=2012. (In case you are working with the development version of Isabelle, the file is simply ~/.isabelle/etc/components). You may need to create this file.

You can now refer to article ABC from the AFP in some theory of yours via

imports "$AFP/ABC"

This allows you to distribute your material separately from any AFP theories. Users of your distribution also need to install the AFP in the above manner.

Example: you are using Isabelle2012, and have downloaded your afp directory to ~/afp. You would add the line ~/afp to the file ~/.isabelle/components.
